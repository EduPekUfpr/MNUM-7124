Readme

Todos os arquivos seguem a mesma estrutura:

Dois escalares que são as linhas e colunas da matriz A

N1 - # Linhas
N2 - # Colunas

Vetor custo de tamanho N2 

Matrix A com N1 Linhas e N2 Colunas

Vetor "b" ou RHS (Right hand side) de tamanho N1

Todos os valores são float com 4 casas decimais

Os nomes dos arquivos têm o seguinte formato "Knap_C" + N1 + "I" + N2 + ".dat"
